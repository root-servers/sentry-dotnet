    Parameters
    Type Name Description
    BaseScope from

    The scope to data copy from.
    BaseScope to

    The scope to copy data to.
    Remarks

    Applies the data of 'from' into 'to'. If data in 'from' is null, 'to' is unmodified. Conflicting keys are not overriden. This is a shallow copy.
